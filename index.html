Known HTML 5 tags

  • a - anchor
  • p - paragraph
  • br - line break
  • div - division
  • pre - preformated content block element
  • code - written code (inline in texts)
  • section - group of associated elements
  • article - a typical article
  • nav - a main navigation
  • footer - a footer block
  • header - a header block
  • aside - in an article related content and in other environments secondary content
  • img - image
  • ul - unordered list
  • ol - ordered list
  • li - list element
  • h1-h6 - headline elements
  • em - emphasize text
  • strong - make text strong
  • span - separate text element
  • table
  • thead - table head
  • th - table head cell
  • tbody - table body
  • tr - table row
  • td - table data

The anchor tag

The anchor tag has multiple attributes. We know: href, style, class, id
